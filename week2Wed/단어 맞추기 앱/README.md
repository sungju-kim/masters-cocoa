# 단어 맞추기 앱

> ### 기본동작 
>> * 현재 입력해야할 단어, 다음에 입력할 단어를 표시한다.
>> * 현재 입력한 단어가 입력해야할 단어와 일치하면 다음단어로 넘어간다.
>> * 현재 입력한 단어가 입력해야할 단어와 다르면 입력해야할 단어를 빨간색으로 표시한다.
>> * 다음에 입력할 단어가 없으면 빈 값을 출력한다.
>> * 실행할때마다 단어의 순서는 바뀐다.
