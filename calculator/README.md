# 계산기 앱

> ### 기본동작 
>> * 계산기능 구현
>> * AC 또는 흔들었을때 스택 초기화
>> * 숫자와 연산자를 입력할때마다 스택 적용
>> * = 를 입력하여 연산이 끝나면 다음 스택에 적용
>> * stack5까지 입력 가능하며 초과할시 stack1 부터 순차 적용
