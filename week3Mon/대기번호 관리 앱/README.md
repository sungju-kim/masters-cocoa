# 대기번호 관리 앱

> ### 기본동작 
>> * 대기 버튼을 누르면 현재 대기인원이 올라간다.
>> * 대기인원은 최대 20명까지이며 20명이 대기중일땐 빨간색으로 표기한다.
>> * 대기인원이 20명일때 대기버튼을 비활성화 한다.
>> * 대기인원이 1명 이상일때 완료 버튼이 활성화 된다.
>> * 화면가운데 가능 인원중 대기인원 비율을 표시한다.
